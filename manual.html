PASO A PASO DE CÓMO SE HIZO MI PAGINA WEB

Paso 1: Creación de la carpeta de trabajo

Lo primero que hice fue crear una carpeta nueva en el Escritorio de mi computadora para guardar todos los archivos juntos y que los enlaces no se rompieran.

Paso 2: Creación de la página principal (index.html)

Abrí el Bloc de Notas y creé el archivo index.html. Aquí usé las etiquetas <frameset> y <frame> para dividir la pantalla en dos columnas: un lado izquierdo para el menú y un lado derecho para ver el contenido.

Paso 3: Creación del menú (menu.html)

Creé el archivo menu.html para poner los botones de navegación en el lado izquierdo. Usé la etiqueta <a> para hacer los enlaces y les puse el atributo target="derecha" para que al apacharlos se abran en el marco del lado derecho.

Paso 4: Pantalla de bienvenida (inicio.html)

Para que la pantalla de la derecha no empezara vacía con un cuadro feo, hice una página de inicio con un saludo y le metí una imagen usando la etiqueta <img>. Luego configuré el index.html para que cargara esta página por defecto.

Paso 5: Integración del Currículum y la Galería

Guardé mi archivo de hoja de vida (curriculum_cris.pdf) y el archivo de las fotos (galeria.pdf) adentro de la misma carpeta del Escritorio para que el menú los pudiera abrir directamente.

Paso 6: Creación de la tabla de notas (notas.html)

Hice el archivo notas.html para mostrar mis calificaciones. Para esto usé etiquetas de tablas básicas que son <table>, <tr> para hacer las filas para abajo, y <td> para escribir adentro de los cuadritos.

Paso 7: Módulo de video (video.html)

Creé la página para reproducir el video de fútbol. Usé la etiqueta nativa <video> con sus controles para que se pueda pausar y reproducir de forma local sin usar internet.

Paso 8: Programación del juego (juego.html)

Hice el juego de la trivia de cultura general de 4 preguntas. Aquí usé botones normales de HTML y abrí la etiqueta <script> para meter el código de JavaScript. Programé funciones para que al hacer clic revise si es correcto, sume puntos y pase sola a la siguiente pregunta usando un tiempo de espera.

Paso 9: Documentación final (manual.html)

Por último, creé este archivo que estás leyendo para dejar por escrito cada paso que seguí para terminar la tarea de la universidad.
